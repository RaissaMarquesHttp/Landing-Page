    Áreas

    Front End

    O desenvolvimento de front-end aborda o lado voltado para o usuário de um site. Essas tecnologias incluem linguagens de computador como JavaScript, CSS e HTML.

    Back End

    Back End é o desenvolvimento de recursos e estrutura por trás da programação, que garante a realização das ações por parte dos usuários em um computador.

    Design UX

    A sigla UX vem de "User Experience", que traduzido, nos dá "Experiência do Usuário". Quando falamos em UX Design, nos referimos ao design focado na experiência de quem usa um determinado produto ou serviço.

    Analise de Dados

    Análise de dados é o processo de aplicação de técnicas estatísticas e lógicas para avaliar informações obtidas a partir de determinados processos. O principal objetivo da prática é extrair informações úteis a partir dos dados.
